### Introduction
This application could be used to process any csv file with a format comprising of four columns of data types String, String, Integer, and String. The fields for these data types are Name, Date of birth, Annual Salary, and Role.
Specifically, this application takes the input file and calculation type as arguments using the command line. After parsing the file, it finds the following:
a. Total salary in dollars.
b. Total salary in dollars grouped by role.
c. Total salary in dollars by time unit (hour, week, month).
d. Total salary in dollars by time unit grouped by role.
e. Grouping of people whose annual salary are as close to teh input salary L without going over L. Hence, the parameter is salary limit L. The output is a group of employees.
The output is printed to terminal.
